# Lab0Git
Подсчет уникальных элементов массива

Цель: Реализовать алгоритм для определения количества чисел, которые встречаются в заданном целочисленном массиве ровно один раз.

Алгоритм решения:

1) Поиск эталона:
Найти максимальное значение в массиве, чтобы вычислить «метку» (max + 1), которая гарантированно отсутствует в исходных данных.

2) Обработка дубликатов:
Сравнить каждый элемент с последующими. Если найдены совпадения, пометить все повторные вхождения специальной меткой.

3) Фильтрация:
Если для текущего элемента были найдены дубликаты или он сам является ранее помеченным дубликатом, он исключается из подсчета.

4) Результат:
Вывести итоговое количество элементов, оставшихся уникальными после полной проверки массива.
